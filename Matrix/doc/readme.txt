Program ini dibuat agar dapat menyelesaikan sistem persamaan linear dan interpolasi.
Cara menggunakan program ini adalah dengan menginstall JDK agar dapat di-compile.
Saat sudah berhasil dicompile dan di run,akan keluar menu pilihan berdasarkan kebutuhan.
Pilihan 1 dan 2 digunakan untuk menyelesaikan Sistem Persamaan Linear.
Pilihan 3 digunakan untuk menyelesaikan Sistem Persamaan Linear khusus yaitu Matriks Hilbert.
Pilihan 4 dan 5 digunakan untuk interpolasi yang bertujuan untuk menghampiri sebuah fungsi atau data.
Dokumentasi program ada pada laporan yang terlampir kemudian.
